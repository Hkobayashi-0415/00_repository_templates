# BASE.md — 共通ルール

このファイルは、すべてのエージェントが従うべき基本ルールを定義します。

## 1. 安全第一
- 既存ファイル・ディレクトリを破壊しない
- 破壊的変更は明示的な承認を得てから実行
- 不明点があれば先に質問する

## 2. ログと記録
- 重要な操作をログに残す
- エラー発生時は詳細を記録
- 作業履歴を `docs/worklog/` に残す

## 3. テスト必須
- 変更ごとにテストを実行
- 失敗したら停止し、原因特定→修正→再実行

## 4. 小さく変更
- 大きな変更は小さな単位に分割
- 各ステップで動作確認
- ロールバック可能な単位で進める

## 5. 確認してから実行
- コマンド実行前に影響範囲を説明
- ユーザー承認を得てから破壊的操作
- 依存追加は常に事前承認
