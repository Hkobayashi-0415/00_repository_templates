# Daily Setup

作業開始時に実行するチェックリストです。

## 1. ブランチ確認
```bash
git branch --show-current
git status
git pull origin main
```

## 2. タスク確認
- [ ] 今日のタスクを確認
- [ ] 優先順位を決定
- [ ] ブロッカーがないか確認

## 3. 環境確認
- [ ] 開発サーバーが起動するか
- [ ] テストが通るか
- [ ] 依存関係が最新か

## 4. 作業開始
- [ ] 作業ブランチを作成（必要な場合）
- [ ] worklog に作業開始を記録
