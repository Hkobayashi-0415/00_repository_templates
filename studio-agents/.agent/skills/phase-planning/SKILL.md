# Phase Planning

## Purpose
- プロジェクトのフェーズを計画し、マイルストーンとタスクを定義する

## When to Use
- 新規プロジェクト開始時
- フェーズ切り替え時

## Inputs
- プロジェクト概要・ゴール
- 利用可能リソース
- 期限・制約条件

## Output Format
- フェーズ定義書
- マイルストーン一覧
- タスクブレークダウン

## Steps
1. プロジェクトゴールを明確化
2. 主要フェーズを定義
3. マイルストーンを設定
4. タスクを洗い出し
5. スケジュールを作成

## Checks (DoD)
- [ ] 全フェーズが定義されている
- [ ] マイルストーンに完了条件がある
- [ ] タスクは実行可能な粒度

「新機能リリースに向けて4週間のフェーズ計画を作成してください」
